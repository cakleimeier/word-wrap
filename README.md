# Word Wrap Challenge
This is an example solution to the Coding Dojo [Word Wrap Challenge](https://codingdojo.org/kata/WordWrap/).

It's built on Laravel and Vue. To demo the application:
- Download this repository
- Navigate to it in Terminal
- Enter `php artisan serve`

The logic is in `/app/Http/Controllers/Controller.php`
The Vue components are in `/resources/js/components/FormOutput.vue`
